# Project_Churn_Organization

Prever quais clientes deixarão de usar a organização analisando alguns atributos importantes e aplicando Machine Learning e Deep Learning.

A rotatividade de clientes refere-se ao momento em que um cliente (jogador, assinante, usuário, etc.) encerra seu relacionamento com uma empresa. Empresas online geralmente consideram um cliente como tendo cancelado o serviço após um determinado período desde sua última interação com o site ou serviço.

Um Modelo Preditivo de Rotatividade é uma ferramenta que define as etapas e estágios da rotatividade de clientes, ou seja, quando um cliente abandona seu serviço ou produto. ... Mas com um modelo de rotatividade em constante evolução, você pode lutar pela retenção agindo com base nas métricas à medida que elas acontecem.

A rotatividade de clientes ocorre quando clientes ou assinantes param de fazer negócios com uma empresa ou serviço, também conhecida como perda de clientes. Também é chamada de perda de clientes. ... Semelhante ao conceito de previsão de rotatividade de funcionários, vamos prever a rotatividade de clientes usando um conjunto de dados de telecomunicações.
